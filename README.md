# Terracotta Bistro

A modern restaurant website with a React frontend and Express backend.

## How to Run

You will need two terminal windows.

### 1. Start the Backend

```bash
cd backend
npm install
npm run dev
```

The server will start on `http://localhost:4000`.

### 2. Start the Frontend

Open a new terminal window:

```bash
cd terracotta-ui
npm install
npm run dev
```

The app will open at `http://localhost:5173`.
